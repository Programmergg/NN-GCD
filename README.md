## 🧩 Overview

This repository provides the official implementation for:

**A Fresh Look at Generalized Category Discovery through Non-negative Matrix Factorization**

Generalized Category Discovery (GCD) aims to classify both base and novel images using labeled base data.  
In this work, we revisit GCD from a **non-negative matrix factorization** perspective and propose **NN-GCD**, a simple yet effective framework that:

- reformulates the optimization of the co-occurrence matrix as a non-negative contrastive learning problem,
- bridges **optimal K-means**, **Symmetric Non-negative Matrix Factorization (SNMF)**, and **non-negative contrastive learning**,
- introduces tailored optimization components to satisfy non-negative constraints and encourage sparse structure,
- achieves strong performance on standard GCD benchmarks.

## ✨ Highlights

- A fresh theoretical perspective on **Generalized Category Discovery**

- A unified view connecting **K-means**, **SNMF**, and **contrastive learning**
- Hybrid sparse regularization for improving base-novel separation
- Strong empirical performance on semantic shift benchmarks
